# ATTENDX 🚀

A futuristic smart attendance management system built using:

- Python (Flask)
- HTML
- CSS
- JavaScript
- C Language

ATTENDX helps teachers manage student attendance with a modern dashboard interface and real-time attendance analytics.

---

# ✨ Features

✅ Add Students Once  
✅ Daily Attendance Marking  
✅ Individual Attendance Percentage  
✅ Modern Glassmorphism UI  
✅ Real-Time Attendance Analytics  
✅ Present / Absent Toggle Buttons  
✅ Flask Backend Integration  
✅ Responsive Dashboard  
✅ Live Attendance Records  
✅ Beginner Friendly Project Structure
